## Status

The WitnessChain location proof plugin is being developed on the [`develop`](https://github.com/location-proofs/plugin-witnesschain/tree/develop) branch.

## About

WitnessChain provides infrastructure-based proof-of-location via network latency triangulation. This plugin will integrate WitnessChain challenges into the Astral SDK location proof framework.

For more information about WitnessChain, visit [witnesschain.com](https://witnesschain.com).
